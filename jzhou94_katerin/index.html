The goal

The goal of this project is to analyze the factors that influence crime rates across districts in the Boston Area.

Narrative

The five datasets are employee earnings in Boston, location of public schools in Boston, crime incidents in Boston, colleges and universities in Boston and street lights in Boston. These datasets can be combined on location to see various statistics relating to crime in different counties of Greater Boston area. Specifically, we want to see the presences of schools on the occurrences and different crimes. Also, we want to see if there is a correlation between wage of police officers and general crime in an area. These questions are interesting because politicians often point to increasing security and education as ways to improve living conditions.

Street Lights

The given dataset contains the coordinates of all street lights in Boston Area.

For the purpose of our project we had to obtain the number of street lights in every district of Boston. In order to retrieve the zip codes, we used a google maps api that takes longitute and latitude and returns a json file with the corresponding address. From this file one can extract a zip code.

First, we found the coordinates of street lights, then we ran an algorithm that adds a zip code value to every entry in the file. Using Map Reduce we managed to obtain the number of street lights for every zip code.

Statistical Analysis

p-value

Definition

The p-value is the probability that a given statistical model would be the same as or more extreme than the actual observed results.

Our Hypothesis

Null Hypothesis: The average earnings of police officers HAS NO effect on the number of crimes. Therefore, the average number of crimes should have no correlation with earnings of police officers.

Alternative Hypothesis: The average earnings of police officers HAS an effect on the number of crimes. Therefore, the average number of crimes should be negatevily correlated with the average earnings of police officers.

Sample data:
Avg earnings: [88222.93, 2369.58, 138971.46, 135889.69]
Number of Crimes: [178.0, 109.0, 118.0, 81.0]

After executing the p-value algorithm, we found a p-value equal to 1. Therefore, we cannot conclude that a significant difference exists between the null and alternative hypotheses.

k-means

Definition

k-means is a simple and easy way to classify a given data set through a certain number of clusters (assume k clusters) fixed a priori.

We used the k-means algorithm in order to find the point with the highest density of crimes.
By traversing the dataset that contains crime incidents and their coordinates, we obtained the value equal to (-71.08306194165799, 42.3237643132037), which is the mean value should there be one cluster for crimes.
